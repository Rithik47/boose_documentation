      AppVariable Class

      Manages variable storage and expression evaluation for the BOOSE language interpreter. Supports integers, reals (doubles), booleans, arrays, and user-defined methods. Provides expression parsers for numeric and boolean expressions with operator precedence.

      Remarks

      This class serves as the central variable storage and expression evaluator. It supports:
      • Integer and real (double) scalar variables
      • Boolean variables with logical operators
      • Integer and real arrays with element-level access
      • User-defined methods with parameter binding
      • Numeric expression evaluation with +, -, *, /, parentheses, and variable references
      • Boolean expression evaluation with &&, ||, !, parentheses, and variable references
      Variable names are case-insensitive. Method names can coexist with variable names to support storing return values.
