      Remarks

      This interpreter supports:
      • Variable declarations (int, real, boolean, arrays)
      • Control structures (if/else, while, for)
      • User-defined methods with parameters and return values
      • Drawing commands (moveto, drawto, circle, rect, pen, write)
      • Numeric and boolean expressions
      • Line continuation for multi-line statements
      The interpreter executes code recursively for nested control structures and method calls.
