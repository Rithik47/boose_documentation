      AppForTryParseBlock Method

      Attempts to parse a FOR loop block starting at the specified index in the list of logical lines. Extracts the loop variable name, start expression, end expression, optional step expression, body statements, and determines the line index after the block ends.

      Parameters

      lines  ListString
      The list of logical source code lines to parse.
      start  Int32
      The zero-based index of the 'for' statement line.
      varName  String
      When this method returns true, contains the name of the loop control variable.
      startExpr  String
      When this method returns true, contains the expression for the loop start value.
      endExpr  String
      When this method returns true, contains the expression for the loop end value.
      stepExpr  String
      When this method returns true, contains the expression for the loop step value, or null if no step clause was specified (defaults to 1).
      bodyLines  ListString
      When this method returns true, contains the list of lines in the loop body (between for and end for).
      nextIndex  Int32
      When this method returns true, contains the index of the line immediately following the 'end for'.

      Return Value

      Boolean
      true if a FOR block was successfully parsed starting at the specified index; false if the line at start is not a 'for' statement.

      Exceptions

      ParserException Thrown when the FOR syntax is invalid (missing '=' or 'to' keyword), or when a matching 'end for' is not found.
